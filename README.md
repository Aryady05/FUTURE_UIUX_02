# Shree Yoga+

Mobile-first appointment booking app for a yoga studio.

## Stack

- Frontend: React Native (Expo)
- Backend: Node.js + Express
- Database: PostgreSQL
- Auth: Firebase Auth
- Notifications: Firebase FCM

## Frontend Features

- Demo-ready onboarding flow
- Home dashboard with featured classes and studio insights
- Explore screen for appointment discovery
- Session detail and booking flow with mock state
- Bookings screen with cancel action
- Profile screen with notification preferences
- Firebase Auth and notification service stubs for backend integration
